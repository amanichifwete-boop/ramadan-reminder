# Ramadan Countdown Reminder

This repository is for automating Ramadan countdown reminders via WhatsApp Business API. It pulls user data from a Google Sheet and sends daily reminders to opted-in recipients with the Hijri and Gregorian dates.

## Initial Setup

1. Create and configure this repository.
2. Integrate Google Sheet for user data.
3. Connect to WhatsApp Business API.

More details to follow in each step.
